## 🚀 Step 1: ローカルサーバーの起動

### 方法A: Python（推奨）

```bash
# ターミナルを開く
cd /Users/iorikawano/Documents/sakura-talk/v4-instagram

# サーバーを起動
python3 -m http.server 8000

# ブラウザで開く
open http://localhost:8000
```

### 方法B: Node.js

```bash
# serve がインストールされていない場合
npm install -g serve

# サーバーを起動
cd /Users/iorikawano/Documents/sakura-talk/v4-instagram
serve -p 8000

# ブラウザで開く
open http://localhost:8000
```

### 方法C: 直接ファイルを開く（簡易）

```bash
# Finderで v4-instagram/index.html を探す
# ファイルをダブルクリックしてブラウザで開く
```

**注意**: 方法Cは一部の機能（フォント読み込みなど）で問題が出る可能性があるため、A または B を推奨

## ⌨️ Step 3: キーボードナビゲーションテスト

### テスト手順

1. **ページをリロード**: `Cmd + R`
2. **Tabキーを押す**: 最初に "Skip to main content" が表示されるべき
3. **Enterキーを押す**: メインコンテンツにジャンプ
4. **Tabキーを連続で押す**: フォーカスが以下の順で移動

```
✓ Skip Link → ナビゲーションリンク → Book Nowボタン
✓ すべてのリンク/ボタンに青い枠（フォーカスリング）が表示
✓ フォーカスが見えない要素にジャンプしない
```

### モバイルメニューのテスト（768px以下）

1. **Tab**でハンバーガーボタンにフォーカス
2. **Enter**または**Space**でメニューを開く
3. **Esc**キーでメニューが閉じる
4. メニュー内で**Tab**キーでナビゲート可能


### よくある問題と解決策

#### 1. フォントが読み込まれない
```
解決策: ローカルサーバーを使用（python http.server）
原因: file:// プロトコルでの制限
```

#### 2. スムーススクロールが動作しない
```
解決策: ブラウザのコンソールでエラーを確認
原因: JavaScriptエラーまたはブラウザ非対応
```

#### 3. モバイルメニューが閉じない
```
解決策: ページをリロード、ブラウザキャッシュをクリア
原因: JavaScriptのイベントハンドラの問題
```

#### 4. 画像がはみ出す
```
解決策: 該当の画面サイズでCSSを確認
原因: メディアクエリの不足または固定幅の使用
```
